## Бот «Ответы Судьбы»

**Telegram-бот "Ответы Судьбы"** - это развлекательный бот, который предоставляет пользователям случайные ответы на вопросы и предсказания. Бот предназначен для тех, кто хочет получить шуточный совет, предсказание будущего или просто развлечься. Он умеет распознавать вопросы и ключевые фразы, отвечает на вопросы «да/нет» и выдаёт мотивационные предсказания из заготовленных списков.

### Возможности
- **Ответ да/нет**: если пользователь задаёт вопрос, бот возвращает один из заготовленных ответов.
- **Предсказание**: если пользователь просит предсказание, бот возвращает одну из фраз из списка предсказаний.
- Распознавание вопросов по наличию знака **"?" или вопросительных слов**.
- Обработка текстовых команд для получения предсказаний.
- Команда **/start** для получения справки.

### Требования.
- Библиотека:
  - Python 3.8 или выше
  - `python-telegram-bot==21.6`

### Как бот понимает, что прислать
- Сообщение считается **вопросом** (и даёт ответ да/нет), если:
  - содержит знак вопроса `?`, или
  - начинается с вопросительных слов: «кто», «что», «где», «когда», «зачем», «почему», «как», «… ли», а также фраз «можно ли», «стоит ли», «нужно ли», «будет ли», «правда ли».
- Сообщение считается **запросом предсказания**, если содержит фразы: «дай мне предсказание», «дай предсказание», «предсказание», «предскажи», «хочу предсказание», либо команда `/предсказание`.
- Для получения ответа да/нет напишите **'Ваш вопрос'** или используйте команду **/ответ**?

### Структура проекта
- `answers.py` — списки ответов и предсказаний, функции выбора случайного элемента.
- `bot.py` — логика Telegram‑бота и маршрутизация сообщений.
- `requirements.txt` — зависимости Python.
- `.gitignore` - игнорируемые файлы.

### Установка
- git clone https://github.com/Valeriyakos/fate_answers_bot
- cd fate_answers_bot
- py -m venv venv
- venv\Scripts\activate
- pip install -r requirements.txt

### Запуск
- py bot.py

Бот запустится и начнёт polling. Остановить — `Ctrl+C` в терминале или `Control+C` на маке

### Примеры использования
- `/start` — бот пришлёт краткую инструкцию.
- Вопрос: «Я успею сдать проект завтра?» → бот ответит «да/нет» одной из фраз.
- Просьба: «Дай мне предсказание» → бот пришлёт предсказание.
